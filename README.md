# Mafia bot

A bot that facilitates playing the [popular mafia game](https://en.wikipedia.org/wiki/Mafia_(party_game)) on IRC.

Still under construction.

Started as a small side project to get familiar with Java 8. Issues, PRs, comments and advice welcome :D

## Setup

1. Clone this repo
2. `cp bot.properties.example bot.properties`
3. Fill in the details for server etc in the `bot.properties` file
4. `./gradlew clean run`
